# FocusLock —— 硬件级番茄时钟

> 用硬件约束代替意志力，通过物理设备帮助用户建立健康的工作-休息节奏。

---

# 一、项目概述

## 1.1 项目背景

当前市面上的番茄钟软件普遍存在以下问题：

- 用户可以随时关闭提醒
- 锁屏提醒容易被忽略
- 软件容易被退出或绕过
- 缺乏物理存在感
- 长时间工作后容易进入“心流失控”状态

FocusLock 希望通过独立硬件设备，实现工作与休息之间的物理级约束。

---

## 1.2 产品定位

FocusLock 是一款桌面生产力硬件设备。

其核心能力不是计时，而是：

**在工作结束时主动干预用户行为，强制进入休息状态。**

---

## 1.3 产品价值

帮助用户建立：

```text
45分钟专注工作
        ↓
30秒提前提醒
        ↓
自动锁屏
        ↓
5分钟强制休息
        ↓
重新开始工作
```

循环机制。

# 三、核心功能

## 3.1 番茄钟管理

默认配置：

| 参数 | 默认值 |
|--------|--------|
| 工作时间 | 45分钟 |
| 休息时间 | 5分钟 |
| 提前提醒 | 30秒 |

支持用户自定义。

---

## 3.2 状态机

系统状态：

```text
WORK
工作中

WARNING
即将休息

REST
休息中

PAUSE
暂停

ADMIN
管理员模式
```

状态流转：

```text
WORK
  ↓
WARNING
  ↓
REST
  ↓
WORK
```

---

## 3.3 强制锁屏

工作时间结束后：

### Windows

```text
Win + L
```

### macOS

```text
Ctrl + Cmd + Q
```

### Linux

```text
Ctrl + Alt + L
```

通过 USB HID 键盘模拟实现。

无需安装任何软件。

---

## 3.4 强制休息机制

进入休息模式后：

```text
每10秒
重新发送一次锁屏命令
```

防止用户立即解锁继续工作。

---

## 3.5 状态灯提示

使用 RGB LED。

### 工作中

绿色常亮

### 即将休息

黄色呼吸灯

### 休息中

红色常亮

### 暂停

蓝色常亮

---

## 3.6 蜂鸣器提醒

### 工作结束前30秒

```text
滴
滴
滴
```

### 锁屏时

```text
滴滴滴滴滴
```

### 休息结束

```text
滴——
```

---

## 3.7 OLED 显示

显示内容：

```text
23:15

○ 圆形进度环
```

### 休息模式

```text

04:31

○ 圆形进度环
```

## 3.8 数据统计

记录：

- 今日完成番茄数
- 今日专注时长
- 累计专注时长
- 累计休息时长

## 3.9 管理员模式

长按按钮 5 秒进入。

适用于：

- 会议
- 演示
- 直播
- 特殊工作场景

暂停当前番茄周期。
