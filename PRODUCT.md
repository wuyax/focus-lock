# Product

## Register

product

## Users
Programmers, AI engineers, product managers, designers, freelancers, and prolonged computer users. They access this UI briefly to configure their physical FocusLock device, usually from a mobile phone connected to the device's captive portal Wi-Fi.

## Product Purpose
A local configuration dashboard for FocusLock, a hardware pomodoro timer that physically prevents overworking by locking the user's screen. The web UI allows users to adjust timer intervals, shortcuts, and device behaviors without installing an app. Success is a fast, reliable, and frictionless configuration experience.

## Brand Personality
Modern, clean, technical.

## Anti-references
Not like a generic router admin page. Not cluttered or overly colorful.

## Design Principles
- **Function over form**: The interface should be clear and configuration options instantly accessible.
- **Hardware connection**: The UI should feel like a reliable extension of a solid, modern hardware device.
- **Frictionless**: Users want to set their configuration and get back to work. Minimize clicks and visual noise.

## Accessibility & Inclusion
- High contrast for readability in various lighting conditions.
- Touch-friendly sizing for form inputs and buttons (since it's often accessed via mobile captive portal).
- Clear error and success states.
